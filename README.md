# Semantic Web & Knowledge Graph Technologies

This repository contains MSc coursework projects in Knowledge Technologies, focusing on Semantic Web technologies, RDF knowledge graphs, SPARQL querying, OWL reasoning, ontology modelling, GraphDB, Protégé, and GeoSPARQL.

The projects demonstrate practical and theoretical work on structured knowledge representation, semantic data querying, ontology-based reasoning, and geospatial knowledge graph analytics. These skills are relevant to FAIR data infrastructure, metadata harmonization, semantic interoperability, scientific data integration, and AI-ready data systems.

## Project Overview

The repository includes three main assignments:

### Homework 1: RDF, SPARQL, DBpedia, GraphDB, and schema.org

This assignment focuses on querying RDF knowledge graphs using SPARQL 1.1. It includes:

- DBpedia querying through the public SPARQL endpoint
- Linked Data exploration
- SPARQL queries over Greek administrative geography data
- Use of the Kallikratis administrative geography ontology
- GraphDB-based RDF querying
- schema.org ontology querying
- RDFS inferencing and non-inferencing query formulations

### Homework 2: Description Logics, OWL 2, Protégé, and Reasoning

This assignment focuses on formal knowledge representation and automated reasoning. It includes:

- Translation of natural-language statements into ALCQO and first-order logic
- Construction of ALCQ knowledge bases
- TBox and ABox modelling
- Tableau-style reasoning
- OWL 2 ontology formalization
- Protégé-based reasoning using the HermiT reasoner
- Analysis of inferred class memberships and entailments

### Homework 3: GeoSPARQL and Geospatial Knowledge Graphs

This assignment focuses on geospatial semantic querying using GeoSPARQL. It includes:

- Spatial queries over RDF knowledge graphs
- GeoSPARQL topological relations
- Distance-based querying
- Nearest-location search
- Spatial joins between entities and locations
- Bounding box computation
- District, lake, river, and adjacency queries over geospatial data
- Discussion of practical limitations in GeoSPARQL engines and dataset interoperability

## Included Files

- `HW_1_Kritsotakis_MscDSIT_Knowledge_Technologies.pdf`: Report containing SPARQL queries, explanations, and results for DBpedia, Greek administrative geography data, and schema.org.
- `HW_2_Kritsotakis_MscDSIT_Knowledge_Technologies.pdf`: Report containing Description Logic formalizations, ALCQ/ALCQO knowledge bases, OWL 2 modelling, tableau reasoning, and Protégé/HermiT reasoning results.
- `HW_3_Kritsotakis_MscDSIT_Knowledge_Technologies.pdf`: Report containing GeoSPARQL queries, geospatial reasoning tasks, query results, and discussion of spatial data interoperability limitations.
- `GreekAdministrativeGeographyOntology.rdf`: RDF ontology file used for modelling Greek administrative geography.
- `ontologyDescription.pdf`: Documentation of the Greek administrative geography ontology.
- `pokemon_data.nt`: RDF knowledge graph data used in the GeoSPARQL assignment.
